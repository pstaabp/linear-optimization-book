Problem 3.3.4.

Apply Phase I of the Simplex Method to the following LOP.
\begin{equation*} \begin{aligned} \text{Minimize}\amp \amp w \amp = 12x_1 + 15x_2 \\ \text{s.t.}\amp \amp -3x_1 + 2x_2 \amp \leq 24 \\ \amp \amp 5x_1 -4x_2 \amp \leq -30 \\ \amp \amp x_1 + x_2 \amp \leq 10 \text{and} \amp \amp x_1,\; x_2 \amp \geq 0. \end{aligned} \end{equation*}
Solution.
If we switch the objective function to \(z = -12x_1-15x_2\text{,}\) then the problem is in standard form and we can write the tableau as
\begin{equation*} \left[\begin{array}{rr|rrr|r} -3 \amp 2 \amp 1 \amp 0 \amp 0 \amp 0 \amp 24\\ 5 \amp -4 \amp 0 \amp 1 \amp 0 \amp 0 \amp -30\\ 1 \amp 1 \amp 0 \amp 0 \amp 1 \amp 0 \amp 10 \\ \hline 12 \amp 15 \amp 0 \amp 0 \amp 0 \amp 1 \amp 0\\ \end{array} \right] \end{equation*}
We use phase I of the Simplex Method because there are negatives in the last column. We choose the second column because it is the negative most number in that column (always ignoring the objective value).
\begin{equation*} 2 \mapsto 4,\qquad \left[\begin{array}{rr|rrrr|r} -2 \amp 0 \amp 4 \amp 2 \amp 0 \amp 0 \amp 36\\ -5 \amp 4 \amp 0 \amp -1 \amp 0 \amp 0 \amp 30\\ 9 \amp 0 \amp 0 \amp 1 \amp 4 \amp 0 \amp 10\\ \hline -123 \amp 0 \amp 0 \amp -15 \amp 0 \amp 4 \amp 450\\ \end{array} \right] \end{equation*}
Looking at the tableau above, there is still a negative number in the last column. Itâ€™s in the first row. Looking across the first row, the largest positive ratio uses the first column. Therefore we perform a matrix pivot about the first row, first column.
\begin{equation*} 1 \mapsto 3, \qquad \left[\begin{array}{rr|rrr|r} 2 \amp 0 \amp -4 \amp -2 \amp 0 \amp 156\\ 0 \amp 2 \amp -5 \amp -3 \amp 0 \amp 210\\ \hline 0 \amp 0 \amp -123 \amp -69 \amp 2 \amp 5022\\ \end{array} \right] \end{equation*}
Now that there are no negative numbers in the last column, this tableau is now feasible, completing the first phase of the simplex method.
